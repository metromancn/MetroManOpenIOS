# 地铁通-MetroMan iOS版
本程序介绍了如何从零开始开发一个 [地铁通-MetroMan](https://metroman.cn)

* 开发工具：XCode 11+
* 语言：Swift 5+
* 支持版本：iOS 11 ~ iOS 13
* 系统Lib：PdfKit
* 开源Lib：
  * Alamofire
  * AMap
  * Toast-Swift
  * NVActivityIndicatorView
  * SSZipArchive
